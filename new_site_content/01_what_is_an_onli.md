# Section 001: what is an Onli™

## Copy

- **Headline**: what is an Onli™
- **Subheadline**: hyper-dimensional vector storage objects. a new kind of database that you can use to make things with.

### Content Block
1. **Onli™**
   - Onli™ makes it simple to create digital assets that can’t be copied, faked, or lost. Each asset, called a Genome, is a self-contained object that holds both the data and its living history. Every Genome is cryptographically unique, evolving into a new version whenever it’s updated or transferred — so there’s only ever one authentic copy in existence. Genomes can also carry metadata and vector embeddings, making them powerful for AI, search, and secure documentation management.
   - **Button**: Get Started

2. **Appliances**
   - An Onli™ Powered Appliance is a client or server application built by developers using Onli™ Cloud APIs. Appliances don’t store or process the content of Genomes; instead, they manage connections between owners, and orchestrate transactions using Onli™ Cloud APIs, and enforce business rules defined by the developer. They act as the interface through which users interact with their assets.
   - **Button**: Get Started

3. **Onli™ You**
   - Onli™ You is a free application (available on mobile or desktop) that allows owners to interact with their assets stored in their Vaults. It includes an authenticator for secure identity management. Onli™ Vaults are secure storage containers within the Onli™ system where Assets are held. Each Vault is associated with a Gene credential and provides isolated, tamper-resistant storage for non-fungible Assets that function as micro-currencies or commodities.
   - **Button**: Get Started

4. **Onli™ Cloud**
   - Onli™ Cloud is the management layer for true digital ownership. It’s where developers build Appliances — applications that interact with assets — and where owners manage the functions tied to their hyperdimensional containers. Assets themselves live and move on Onli™ One, the peer-to-peer network that enforces possession. Only the Owner can execute a transfer; Appliances can request actions, but they can never move an asset on their own. With Onli™ Cloud, developers innovate, Appliances orchestrate, and Owners stay fully in control.
